# Comandos basicos de git

## Practica rapida diaria para recordar comandos de GIT

1. Crea directorio ```mkdir praticas_git```

2. Crear archivo ```touch archivo_practica.txt```

3. Anexar texto al archivo:

«Nombre: Ricardo Carvajal
Edad: 38

Profeción: desarrollador de software

a
b
c»

4. Añade archivo al **stage** ```git add archivo_practica.txt``` o ```git add .```

5. Elimina archivo del **stage** ```git rm --cached archivo_practica.txt``` o ```git rm --cached *.*```

6. Añade archivo al **stage** ```git add archivo_practica.txt``` o ```git add .```

7. Gravar en repositorio los cambios con ```git commit -m ""```

8. Ahora hacer una modificacion en el archivo y añdelo al stage y gravalo en el repositorio

9. Ahora realiza el paso 8 nuevamente

10. Ahora ejecuta ```git show``` y ```git log``` para ver la historia. Tambien haz un ```git diff mommit_viejo commit_nuevo``` para ver la comparacion de cambios al estilo del ```git show```
