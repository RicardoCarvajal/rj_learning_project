# Proyectos de prácticas desarrollados durante un aprendizaje

Los proyectos de este repositorio, son proyectos para el aprendizaje de diversas tecnologías de desarrollo de software, solamente han sido creados para probar o practicar un código o framework específico.

## Estructura del repositorio

**general_bases_bash:**  Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación bash

**general_bases_java:** Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación Java

**general_bases_javascript:** Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación JavaScript

**general_bases_python:** Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación python


**basic_projects:**

	1. spring_boot_data_jpa Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot Data JPA

	2. spring_boot_general Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot

	3. spring_boot_graphQL Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot GraphQL con Spring Boot Data JPA 

	4. spring_boot_reactive_unit_test Este proyecto muestra cómo hacer pruebas unitarias en un servicio Spring Boot Reactivo, tanto para las rutas como para los casos de uso 

	5. spring_boot_security Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot Security

	6. spring_boot_vaadin_jpa Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot con Vaadin y Spring Boot Data JPA 

    7. aws_cloudformation_nested_stack Este pequeño proyecto contiene un ejemplo sencillo de como desarrollar infraestructura como codigo en AWS CloudFormation, el proyecto sirve para desplegarlo y construir una VPC con todas las configuraciones necesarias y luego usa esa VPC para crear una base de datos RDS (MySQL) sencilla
