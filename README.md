# Proyectos de prácticas desarrollados durante un aprendizaje

Los proyectos de este repositorio, son proyectos para el aprendizaje de diversas tecnologías de desarrollo de software, solamente han sido creados para probar o practicar un código o framework específico.

## Estructura del repositorio

**general_bases_bash**  Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación bash

**general_bases_java** Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación Java

**general_bases_javascript** Códigos puntuales, que muestran alguna funcionalidad o uso del lenguaje de programación JavaScript


**basic_projects**
|
|_______spring_boot_data_jpa__ Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot Data JPA
|
|_______spring_boot_general__ Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot
|
|_______spring_boot_graphQL__ Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot GraphQL con Spring Boot Data JPA 
|
|_______spring_boot_reactive_unit_test__ Este proyecto muestra cómo hacer pruebas unitarias en un servicio Spring Boot Reactivo, tanto para las rutas como para los casos de uso 
|
|_______spring_boot_security__ Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot Security
|
|_______spring_boot_vaadin_jpa__ Este proyecto contiene las bases de cómo se implementa un servicio de Spring Boot con Vaadin y Spring Boot Data JPA
